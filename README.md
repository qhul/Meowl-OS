# Meowl-OS
*Meowl-OS is a meme operating system based on the iconic Meme-Meowl.*

*Note: Meowl-OS will be a long-term project and will be developed over time.*
*What is Meowl-OS? It’s a lightweight operating system that can be used on both older and modern computers.*

    Meowl-OS is a work in progress
